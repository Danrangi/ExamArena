ExamArena

Developed at Suleja Digital Innovation Hub

About the System

ExamArena is a robust, Flask-based Computer Based Test (CBT) practice application designed to simulate Nigerian examination environments (JAMB, WAEC, NECO). It provides a realistic "War Room" exam interface for students and a comprehensive management panel for administrators.

Key Features

  • **"War Room" Exam Interface:** A distraction-free environment that mimics actual CBT software.
  • **JAMB Simulator:** Simultaneous testing of 4 subjects with a 120-minute strict timer and a built-in on-screen calculator.
  • **Detailed Results & Corrections:** Provides performance scores, visual progress, and correction mode with explanations.
  • **Secure Admin Panel:** Allows full management of subjects and questions (CRUD) with bulk CSV upload functionality.
